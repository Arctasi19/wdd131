Python Card Games

See More ▼

This project showcases my skills in Python programming through the development of various card games. I am actively updating and working on this project, using the skills I learn in my education to improve and build upon it. Current features include functional Blackjack and a basic rendition of Poker. For Blackjack, the game will randomly provide the user with a hand of cards. The user will then have to decide whether to hit (take another card) or stand (keep their current hand). The goal is to get as close to 21 as possible without going over. The computer will also have a hand of cards which the user must beat to win.
